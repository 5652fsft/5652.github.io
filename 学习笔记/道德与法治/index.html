  1. 与个人的人生目标紧密相连。

  1. 在中国的家庭文化中，“孝” 是重要的精神内涵。

# 孝亲敬长的做法

  1. 尽孝在当下，从现在开始，用行动表达孝敬之心。
  2. 学会尊敬、倾听和感恩。

# 亲子间发生碰撞的表现、原因和影响

# 表现：矛盾心理
  1. 一方面，我们希望父母不要过多干涉我们的生活，我们审视父母给予我们的爱，开始质疑父母，甚至挑战父母的权威与经验；
  2. 另一方面，面对父母的信任和放手，有时我们又觉得失落和不安，期望得到更多的关注和呵护。
# 原因：代沟

作为两代人，我们与父母在心智、学识、经历等方面差异较大，对问题的理解、感受等方面必然存在差异，亲子冲突有时难以避免。

# 影响

如果处理不好，亲子冲突就会伤害双方的情感，影响家庭的和睦。

# ★如何化解与父母的冲突

  1. 爱是需要呵护的，学会互动沟通的技巧和掌握应对冲突的智慧。选择不伤害父母感情和亲子关系的做法。

  2. 可以试着去接纳父母的做法，理解父母行为中蕴含的爱。尝试让父母了解我们的变化和需要，用他们能接受的方式表达我们的爱。

  3. 以良好的心态面对家庭的变化，才能让亲情更浓、家庭更和睦。

# 生命

# ★生命的特性，为什么要热爱生命

  1. 生命来之不易。

  2. 生命是独特的。

  3. 生命是不可逆的。

  4. 生命是短暂的。

  5. 每个人都无法抗拒生命发展的自然规律。

# 如何理解生命的接续、生命可以永恒

  1. 身体生命：一代又一代的个体生命实现了人类生命的接续，在人类生命的接续中，我们总能为自己的生命找到一个位置，担当一份使命。
  2. 社会生命：每个人的生命不仅仅是 “我” 的生命，还是 “我们” 的生命。
  3. 精神生命：在生命的接续中，人类的精神文明也不断积累和丰富。每个人不仅要在身体上接续祖先的生命，也要在精神上不断继承和创造人类的文明成果。

# ★为什么要敬畏生命

  1. 我们感受到生命是脆弱的、艰难的，又体会到生命是坚强的、有力量的、崇高的、神圣的。

  2. 生命价值高于一切，生命至上。

# 生命至上的含义与要求

  1. 生命至上，并不意味着只看到自己生命的重要性，也必须承认他人的生命同样重要。
  2. 要自觉地珍爱他人的生命，如同珍爱自己的生命一样。

# ★如何做到敬畏生命

  1. 意识上：敬畏生命，从对自己生命的珍惜走向对他人生命的关怀，意识到每个人都需要与他人共同生活。

  2. 行为上：不漠视自己和他人的生命，谨慎地对待生命关系、处理生命问题，才会尊重、关注、关怀、善待身边的每一个人。

  3. 态度上：对生命的敬畏并不是谁的命令，而是内心的自愿选择。要发自内心地敬畏生命。

# 爱护身体

  1. 需要关注自己的身体状況，养成健康的生活方式。
  2. 需要关注自己的内在感受，学会爱惜自己的身体。
  3. 需要增强安全意识、自我保护意识；提高安全防范能力，掌握一些基本的自救自护方法。

# 养护精神

  1. 需要过精神生活，需要满足精神需求。
  2. 需要协调好物质支持和精神发育的关系。
  3. 需要用优秀传统文化滋养，发扬民族精神。

# 物质支持和精神发育的关系

  1. 精神发育需要物质的支持，但不完全受物质生活条件和外部环境的制约。
  2. 物质贫乏、外部环境艰苦时只要守住心灵，依然可以看到真善美。

# 挫折的含义

  1. 生活的阻碍、失利和失败，就是人们常说的挫折。

  2. 人们对挫折的认识和态度不同，会产生这些不同情绪感受和行为反应。

# 挫折的影响

# 消极

# 积极
  1. 生活中的挫折是我们生命成长中的一部分。
  2. 得意时，挫折会使我们更清醒，避免盲目乐观、精神懈怠。

  1. 活出自我：能够活出自己的人生，实现自我价值。

  2. 帮助他人：在别人需要时付出自己的爱心，无论事情大小，都能承担起自己的责任。

  3. 生命的意义需要自己发现和创造。

  1. 对自己：拒绝贫乏，充盈生命。

  2. 对他人：拒绝冷漠，关切他人。

  3. 在平凡中创造伟大。

# ★怎样认识生命的 “平凡与伟大”

  1. 伟大在于创造和贡献。
  2. 平凡的生命也可以伟大。
  3. 当我们将个体生命和他人的、集体的、民族的、国家的甚至人类的命运联系在一起时，生命便会从平凡中闪耀出伟大。

# 一致时

# 个人意愿和集体规则产生冲突的原因

一方有不正当或不合理需求或个人和集体的需要不同。

# 如何解决个人意愿与集体规则冲突

  1. 通常会让个人意愿服从集体的共同要求。
  2. 理解集体要求的合理性，反思个人意愿的合理性和实现的可能性。

  1. 个人利益与集体利益本质上是一致的。
  2. 当个人利益与集体利益发生冲突时，应把集体利益放在个人利益之上，坚持集体主义。
  3. 反对只顾自己、不顾他人的极端个人主义。

# ★坚持集体主义的内涵

坚持集体主义，不是不关注个人利益，而是在承认个人利益的合理性、保护个人正当利益的前提下，反对只顾自己、不顾他人的极端个人主义。

# ★怎样让集体和声更美

  1. 尽力做好自己，遵守规则，以保持和声的和谐之美。
  2. 对于集体要求中的不合理因素，可通过恰当的方式表达意见，提出积极的改进建议。
  3. 正确处理个人利益与集体利益的关系。
  4. 正确处理与他人的各种关系。
  5. 识大体，顾大局，不因个人矛盾影响集体利益。

  1. 规范作用：法律规定我们应该享有的权利，应该履行的义务。
  2. 保护作用：法律通过解决纠纷和制裁违法犯罪，惩恶扬善、伸张正义，维护我们的合法权益。

# ~ 社会秩序的含义、内容

社会秩序是社会生活的一种有序化状态，包括社会管理秩序、生产秩序、交通秩序和公共场所秩序等。

# 为什么生活需要秩序

  1. 社会正常运行需要秩序。
  2. 社会秩序是人们安居乐业的保障。

# 社会规则的种类

纪律、道德、法律等。

# 社会规则如何维护秩序，社会规则的作用

  1. 社会规则明确社会秩序的内容。
  2. 社会规则保障社会秩序的实现。

# ★自由和规则的关系

  1. 社会规则划定了自由的边界。自由不是随心所欲，它受到道德、纪律、法律等社会规则的约束。
  2. 公民在行使自由和权利时，不得损害国家的、社会的、集体的利益和其他公民的合法的自由和权利。
  3. 社会规则是人们享有自由的保障。违反规则、扰乱社会秩序的行为应当受到相应的处罚。

# ★如何遵守、维护、改进规则

# 自觉遵守规则
  1. 他律和自律。
  2. 发自内心敬畏规则，将规则作为行动的准绳，内化于心、外化于行。
# 坚定维护规则
  1. 从自己做起，自觉遵守社会规则。
  2. 在保护自身安全的基础上，提醒、监督、帮助他人遵守规则。
# 积极改进规则
  1. 规则不是一成不变的，有的需要制定、废除、修改和完善。
  2. 要积极参与规则的的改进和完善，为新规则的形成建言献策。

  3. 民族：诚信促进社会文明、国家兴旺。

# ★如何践行诚信

  1. 树立诚信意识。信守承诺，主动承担责任。
  2. 运用诚信智慧。既恪守诚实的要求，又尊重他人隐私。
  3. 珍惜个人的诚信记录。

# 守法

# 违法行为

# ★含义

违法行为是指出于过错违反法律、法规的规定，危害社会的行为。

# ★类别

  1. 民事违法行为（侵犯他人民事权利或没有依法履行合同的义务）
  2. 行政违法行为（如：谎报险情、破坏铁路封闭网、殴打他人等行为都是违反治安管理处罚法的行政违法行为。）
  3. 刑事违法行为（犯罪）

民事违法行为、行政违法行为对社会的危害相对较轻微，称为一般违法行为。

刑事违法行为（犯罪）是最严重的一种违法行为。

违法行为都要承担法律责任。

# 预防犯罪

# ★什么是刑法

刑法是惩治犯罪、保护国家和人民利益的有力武器，明确规定了什么行为是犯罪，以及对犯罪应当判处什么样的刑罚。

# ★什么是犯罪

犯罪是具有严重的社会危害性、触犯了刑法、应当受到刑罚处罚的行为。

基本特征：严重危害性、刑事违法性、刑罚处罚性。

# 什么是刑罚

犯罪的法律后果是刑罚，是最严厉的强制性法律制裁方法，分为主刑和附加刑。

# ★加强自我防范，预防违法犯罪

  1. 珍惜美好生活，认清犯罪危害，远离犯罪。
  2. 杜绝不良行为。
  3. 增强法治观念，依法自律，做自觉守法的人。要从小事做起，避免沾染不良习气，自觉遵纪守法，防患于未然。

# 善用法律

# ★遇到侵害依法求助的途径

  1. 遇到法律问题或权益受到侵害时，要及时寻求法律援助，依靠法律维护自己的合法权益。
  2. 通过法律服务机构来维护合法权益，如法律服务所、律师事务所、公证处、法律援助中心等。
  3. 受到非法侵害，可寻求国家的法律救济，如依法到公检法控告举报，还可向人民法院提起诉讼。

# 诉讼

诉讼是处理纠纷、应对侵害最正规、权威、有效的手段，是维护合法权益的最后屏障。

诉讼分为：民事诉讼、行政诉讼（“民告官”）、刑事诉讼。

# ★如何应对违法犯罪

  1. 同违法犯罪行为作斗争，是包括青少年在内的全体公民义不容辞的责任。
  2. 面对违法犯罪时，要善于斗争，在保全自己、减少伤害的前提下，巧妙地借助他人或社会的力量，采取机智灵活的方式，同违法犯罪做斗争。
  3. 要积极弘扬社会主义法治精神，形成守法光荣、违法可耻的观念，做到自觉守法、遇事找法，解决问题靠法，努力做社会主义法治的忠实崇尚者、自觉遵守者、坚定捍卫者。

# 社会责任

# 责任与角色

# ~ 责任的含义和来源

责任是一个人分内该做的事，来自对他人的承诺、职业要求、道德规范、法律规定等。

# ~ 角色与责任的关系

每种角色都意味着承担相应责任。

# 承担责任的表现

  1. 首先要对自己负责。
  2. 要学会感恩，主动关心、帮助和服务他人，在辛苦付出中体验对他人负责的快乐和幸福。

# ★承担责任的意义

  1. 个人角度：只有对自己负责，才能充分挖掘、发挥潜能，才有资格、能力、信心承担起时代和国家所赋予的使命。
  2. 他人角度：只有人人具有责任心，自觉履行应尽的责任，才能共享更加幸福美好的生活。
  3. 社会角度：只有人人认识到自己扮演的角色、承担应尽的责任，才能构建各尽其责、各得其所而又和谐相处的社会。

# 承担责任的代价与回报

  1. 代价：付出时间、精力和金钱，可能因做的不好而受责备甚至处罚。
  2. 回报：有获得物质、精神回报的权利。

# ★怎样做负责任的人

  1. 无论是否自愿选择，都应自觉承担相应责任。
  2. 履行社会责任不计代价和回报，敢于承担责任、敢于担当。
  3. 努力提升自身素质，增强履行责任的能力。

# 奉献社会

# 关爱他人

# ★关爱他人的意义

  1. 他人：关爱传递着美好情感，给人带来温暖和希望，是维系友好关系的桥梁。
  2. 社会：关爱是社会和谐稳定的润滑剂和正能量，有利于形成良好的人际氛围，促进社会文明进步。
  3. 自己：关爱他人，收获幸福。

# ★关爱他人的艺术

  1. 心怀善意。道义上支持、物质上帮助、精神上关怀。
  2. 尽己所能，不分大小，贵在爱心，尽己所能帮助他人，奉献社会。
  3. 讲究策略，考虑他人的内心感受，增强自我保护。

# 服务社会

# ★服务社会对个人成长的意义

  1. 体现人生价值。只有积极为社会做贡献，才能得到人们的尊重和认可，实现自身价值。
  2. 能促进全面发展。在服务社会过程中视野不断拓展，能力不断提升，境界不断提高。

# ★奉献社会的途径

  1. 青年担当责任。在新时代，共青团要组织青年参加改革开放和社会主义现代化建设的实践。
  2. 积极参与社会公益活动。
  3. 热爱劳动，爱岗敬业。

# 维护国家利益

# 国家利益

# ~ 内涵及外延

  1. 国家利益是一个主权国家在国际社会中生存需求和发展需求的总和，包括人口、领土、主权和政权等，关系民族生存、国家兴亡。
  2. 国家利益涉及政治、军事、经济、文化、社会等领域，包括安全利益、政治利益、经济利益、文化利益等。
  3. 国家的核心利益包括国家主权、国家安全、领土完整、国家统一、宪法确立的国家政治制度和社会大局稳定、经济社会可持续发展的基本保障。

# ★与人民利益的关系

  1. 国家利益反映广大人民的共同需求，是人民利益的集中表现，国家利益至上，人民利益高于一切，二者相辅相成。
  2. 维护人民利益离不开国家利益，人民利益只有上升、集中到国家利益，运用国家的工具，才能得到真正的维护。
  3. 国家利益的实现离不开人民，国家利益只有反映人民利益，依靠人民艰苦奋斗，才能得到真正的实现。
  4. 在当代中国，国家利益和人民利益是高度统一的。实现中华民族伟大复兴就是将国家和人民视为命运共同体，将二者利益紧密联系在一起。

# ★树立维护国家利益意识

  1. 心怀爱国之情，牢固树立国家利益至上的观念，以热爱祖国为荣，以危害祖国为耻。
  2. 树立和增强危机意识和防范意识，对危害国家利益、威胁国家生存和发展的行为时刻保持警惕。
  3. 增强维护国家利益的责任感和使命感。用理性、务实、文明的心态合法有序地表达爱国情感，维护国家利益。

# ★如何捍卫国家利益

  1. 无论何时何地，都应着眼长远、顾全大局，以国家利益为重，把国家利益放在第一位。
  2. 为了国家利益，有时需要放弃个人利益甚至献出生命。
  3. 捍卫国家尊严，坚决同一切损害国家利益的行为作斗争。日常生活中自觉遵守道德和纪律，积极维护国家稳定团结的局面。

# 国家安全

# 国家安全的重要性

国家安全与我们息息相关，是国家生存和发展的重要保障，是人民幸福安康的前提。

# ~ 如何坚持总体国家安全观

  1. 以人民安全为宗旨，以政治安全为根本，以经济安全为基础，以军事、文化、社会安全为保障，以促进国际安全为依托，走出一条中国特色国家安全道路。
  2. 面对国家安全形势的新变化，要构建集政治、国土、军事、经济、文化、社会、科技、信息、生态、资源、核安全等于一体的国家安全体系。

# ★如何维护国家安全

# 国家

全面推进国防和军队现代化，坚持党对人民军队的绝对领导。

# ★个人，如何履行维护国家安全的义务

维护国家安全，人人可为。

  1. 维护国家安全是我们的共同责任。要增强国家安全意识，树立国家安全利益高于一切的观念，提高防范意识。
  2. 通过各种方式为维护国家安全贡献智慧和力量。
  3. 认真学习有关国家安全和保密工作的法律法规、规章制度。
  4. 每个公民都应严格遵守有关国家安全的法律规定，积极履行维护国家安全的法定义务，不断提高防范能力。

# 建设美好祖国

# 劳动的意义

劳动是财富、幸福的源泉。我国每一项成就的取得，都是广大人民群众用辛勤劳动、诚实劳动、创造性劳动换来的，中国人民用实干精神创造了今天的辉煌。

# 如何对待不同岗位的劳动者

  1. 每个人所处的岗位不同，从事不同的劳动，但都在为国家和社会做贡献。
  2. 正是无数劳动者艰苦奋斗才成就了今天的美好生活。
  3. 无论是脑力劳动者还是体力劳动者，都是和国家的建设者，都值得尊敬和学习。

# 青少年发扬实干精神

# 原因

中国梦的实现需要一代代人埋头苦干和接力奋斗，需要每个人在各自岗位上付出辛勤和汗水。只有继续发扬实干精神，才能创造新的辉煌。

# 做法

接过历史的接力棒，继续发扬实干精神，努力学习，积极探索，（以执着的信念、优良的品德、丰富的知识、过硬的本领，）担负起历史重任。
