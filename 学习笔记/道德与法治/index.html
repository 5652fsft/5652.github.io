# 1 成长

# 1.1 梦想

# 1. 梦想的含义

梦想是对未来美好生活的愿望，是人类天真无邪、美丽可爱的愿望。

# 2. 梦想的重要性

  1. 编织人生梦想，是青少年时期的重要生命主题。
  2. 梦想能不断激发我们生命的热情和勇气，让生活更有色彩。有梦想，就有希望。
  3. 少年的梦想，虽然总和现实有一定距离，甚至不切实际，但是人类需要这样的梦想，因为有了这样的梦想，才能不断地进步和发展。

# ★3. 少年梦想的特点

  1. 与个人的人生目标紧密相连。
  2. 与时代的脉搏紧密相连，与中国梦密不可分。

# ★4. 怎样实现梦想

  1. 努力是梦想与现实之间的桥梁。
  2. 实现梦想，要付诸行动。
  3. 努力，需要立志、坚持、方法。

# 1.2 学习

# 1. 正确看待学习

  1. 是中学阶段的重要任务。
  2. 包括知识的获取、能力的培养、如何做人。
  3. 不仅局限在学校，可以从一切经历中学习。
  4. 需要自觉、主动的态度。
  5. 伴随成长，没有终点。我们终生都在、都需要学习。

# ★2. 意义

  1. 不仅让我们能够生存，而且让我们拥有更充实的生活。
  2. 学习点亮我们心中的明灯，激发前进的动力，让我们获得成长。
  3. 可以帮助他人，服务社会，为幸福生活奠基。

# 3. 如何体味学习

学习是有苦有乐的。

# ★4. 如何学会学习

  1. 需要发现并保持对学习的兴趣。

  2. 需要掌握科学的学习方法。

  3. 需要善于运用不同的学习方式。

促进自我发展、与他人的交往。

# 1.3.2 做更好的自己

# 1. 接纳自己

# ~ 原因

每个人都是独一无二的个体，性格、兴趣等各有特点。

# 要求
  1. 接纳自己的全部。
  2. 需要乐观的心态、勇气和智慧。

# 2. 欣赏自己

  1. 欣赏自己的独特，优点，努力，为他人的奉献。
  2. 不是骄傲自大，也不是目中无人，而是自我鼓励和自我奋进。
  3. 要善于向他人学习，与他人合作，成为一个内涵更加丰富的人。

# ★3. 怎样做更好的自己

  1. 扬长避短。
  2. 主动改正缺点。
  3. 不断激发自己的潜能。

# 2 友谊

# 2.1 友谊

# ★1. 朋友的重要性

  1. 朋友对一个人的影响很大。
  2. 朋友见证了我们一起走过的成长历程，我们需要真诚友善的朋友，要善交益友，不交损友。
  3. 朋友丰富了我们的生活经验，友谊让我们更深刻地体悟生命的美好。

# 2. 友谊的特质

  1. 是一种亲密的关系。
  2. 是平等的、双向的。
  3. 是一种心灵的相遇。

# ★3. 友谊的澄清

  1. 友谊不是一成不变的。
  2. 竞争并不必然伤害友谊，关键是对待竞争的态度。
  3. 友谊不能没有原则，不能不加分辨地为朋友做任何事。当朋友误入歧途时，如果不予规劝甚至推波助澜，反而会伤害朋友，伤害友谊。

# 2.2.1 建立与维护友谊

# ★1. 建立友谊的做法

开放自己，持续的行动。

# ★2. 应该怎样呵护友谊

  1. 用心去关怀对方。
  2. 尊重对方。
  3. 正确处理冲突。
  4. 正确对待交友中受到的伤害。

# 2.2.2 网上交友

# 1. 网上交往的特点

网上交往具有虚拟、平等、自主等特点。

# ~2. 网上交友的影响

# 积极
  1. 超越时空限制，把天涯海角、素不相识的人连在一起。
  2. 开辟了人际交往的新通道，让我们有更多机会结交新的伙伴，拓展交往圈。

  1. 有时关闭了与他人沟通的心灵之门。

# ★3. 网上交往需要注意什么

# 3 师长情谊

# 3.1 师生

# 3.1.1 走近老师

# 1. 了解老师的职业

  1. 教师是一个古老的职业，作为教育工作者，是人类文明的传承者之一。
  2. 在现代社会，教师已发展为一种专门职业。教师是履行教育教学职责的专业人员，承担教书育人的使命。
  3. 时代发展对教师提出的新要求 “四有教师”—— 有理想信念、有道德情操、有扎实学识、有仁爱之心。

# 2. 如何对待风格不同的老师

  1. 要承认老师之间的差异，接纳每位老师的不同，他们身上都有值得我们学习的地方。
  2. 无论什么风格的老师都应该受到尊重。

# 3.1.2 师生交往

# ★1. 教学相长的基本内涵与实质

教与学是师生相互陪伴、相互促进、共同成长的过程。一方面，我们的学习离不开老师的引领和指导；另一方面，与老师交流互动，也可促进老师更好地” 教”。

# ★2. 与老师交往、建立良好师生关系的与方法

# 正确对待老师的引领和指导

要主动参与、勤学好问；

真诚、恰当地向老师表达自己的观点和见解；

与老师分享自己的学习感受、学习成果。

# ★正确对待老师的表扬和批评

老师的批评意味着关心、提醒和劝诫，可以帮助我们反省自己，改进不足。

对待老师的批评，要把注意力放在老师批评的内容和用意上，理解老师的良苦用心。

# 彼此尊重

尊重老师，是中华民族的优良传统。

在平等相待、相互促进的师生交往中，老师是知识学习的指导者，也是精神成长的引路人。

应该主动关心老师、理解老师。

# 3. 为什么要尊重老师

老师是知识学习的指导者，也是精神成长的引路人。

# ★4. 师生交往的良好状态

学生乐于学习，老师寓教于乐，师生彼此尊重、相互关心、携手共进，这是师生交往的良好状态。

# 3.2 亲情

# 3.2.1 家的意味

# ~1. 家的含义

家庭是由婚姻关系、血缘关系或收养关系结合而成的亲属生活组织。

# 2. 家的作用

  1. 家是我们身心的寄居之所。
  2. 家是我们心灵的港湾。
  3. 在中国人的心目中，家是代代传承、血脉相连的生活共同体，是甜蜜、温暖、轻松的避风港。

# ★4. 为什么要孝亲敬长

  1. 在中国的家庭文化中，“孝” 是重要的精神内涵。

  2. 孝亲敬长是中华民族的传统美德，也是每个中国公民的法定义务。

# 5. 孝亲敬长的做法

  1. 尽孝在当下，从现在开始，用行动表达孝敬之心。
  2. 学会尊敬、倾听和感恩。

# 3.2.2 爱在家人间

# ~1. 亲子间发生碰撞的表现、原因和影响

# 表现：矛盾心理
  1. 一方面，我们希望父母不要过多干涉我们的生活，我们审视父母给予我们的爱，开始质疑父母，甚至挑战父母的权威与经验；
  2. 另一方面，面对父母的信任和放手，有时我们又觉得失落和不安，期望得到更多的关注和呵护。
# 原因：代沟

作为两代人，我们与父母在心智、学识、经历等方面差异较大，对问题的理解、感受等方面必然存在差异，亲子冲突有时难以避免。

# 影响

如果处理不好，亲子冲突就会伤害双方的情感，影响家庭的和睦。

# ★2. 如何化解与父母的冲突

  1. 爱是需要呵护的，学会互动沟通的技巧和掌握应对冲突的智慧。选择不伤害父母感情和亲子关系的做法。

  2. 可以试着去接纳父母的做法，理解父母行为中蕴含的爱。尝试让父母了解我们的变化和需要，用他们能接受的方式表达我们的爱。

# 3.2.3 让家更美好

# ~1. 现代家庭发生变化的表现

  1. 家庭趋向小型化。
  2. 家庭成员的交流、沟通方式发生了较大的变化。
  3. 家庭氛围越来越平等、民主。

# ★2. 怎样建设和谐家庭

  1. 要家庭成员之间相互理解、信任、体谅和包容，可以增进理解，化解矛盾与冲突。
  2. 要积极参与家务劳动，养成劳动习惯，不断提高自我管理能力，增强家庭责任意识，为建设和谐家庭作贡献。
  3. 以良好的心态面对家庭的变化，才能让亲情更浓、家庭更和睦。

# 4 生命

# 4.1 探问生命

# 4.1.1 生命永恒

# ★1. 生命的特性，为什么要热爱生命

  1. 生命来之不易。

  2. 生命是独特的。

  3. 生命是不可逆的。

  4. 生命是短暂的。

  5. 每个人都无法抗拒生命发展的自然规律。

# 2. 如何理解生命的接续、生命可以永恒

  1. 身体生命：一代又一代的个体生命实现了人类生命的接续，在人类生命的接续中，我们总能为自己的生命找到一个位置，担当一份使命。
  2. 社会生命：每个人的生命不仅仅是 “我” 的生命，还是 “我们” 的生命。
  3. 精神生命：在生命的接续中，人类的精神文明也不断积累和丰富。每个人不仅要在身体上接续祖先的生命，也要在精神上不断继承和创造人类的文明成果。

# 4.1.2 敬畏生命

# ★1. 为什么要敬畏生命

  1. 我们感受到生命是脆弱的、艰难的，又体会到生命是坚强的、有力量的、崇高的、神圣的。

  2. 生命价值高于一切，生命至上。

# 2. 生命至上的含义与要求

  1. 生命至上，并不意味着只看到自己生命的重要性，也必须承认他人的生命同样重要。
  2. 要自觉地珍爱他人的生命，如同珍爱自己的生命一样。

# ★3. 如何做到敬畏生命

  1. 意识上：敬畏生命，从对自己生命的珍惜走向对他人生命的关怀，意识到每个人都需要与他人共同生活。

  2. 行为上：不漠视自己和他人的生命，谨慎地对待生命关系、处理生命问题，才会尊重、关注、关怀、善待身边的每一个人。

  3. 态度上：对生命的敬畏并不是谁的命令，而是内心的自愿选择。要发自内心地敬畏生命。

# 4.2 珍视生命

# 4.2.1 守护生命

# 1. 爱护身体

  1. 需要关注自己的身体状況，养成健康的生活方式。
  2. 需要关注自己的内在感受，学会爱惜自己的身体。
  3. 需要增强安全意识、自我保护意识；提高安全防范能力，掌握一些基本的自救自护方法。

# 2. 养护精神

  1. 需要过精神生活，需要满足精神需求。
  2. 需要协调好物质支持和精神发育的关系。
  3. 需要用优秀传统文化滋养，发扬民族精神。

# 3. 物质支持和精神发育的关系

  1. 精神发育需要物质的支持，但不完全受物质生活条件和外部环境的制约。
  2. 物质贫乏、外部环境艰苦时只要守住心灵，依然可以看到真善美。

# 4.2.2 增强生命的韧性

# 1. 挫折的含义

  1. 生活的阻碍、失利和失败，就是人们常说的挫折。

  2. 人们对挫折的认识和态度不同，会产生这些不同情绪感受和行为反应。

# 2. 挫折的影响

# 消极

面对挫折，可能会产生一些负面的情绪感受，如果一味沉浸其中就容易消沉，甚至做出不恰当的行为。

# 积极
  1. 生活中的挫折是我们生命成长中的一部分。
  2. 得意时，挫折会使我们更清醒，避免盲目乐观、精神懈怠。
  3. 失意时，挫折会使我们获得更加丰富的生活经验。

# ★3. 如何发掘生命的力量应对挫折

  1. 增强生命的承受力、自我调节和自我修复的能力。

  2. 培养自己面对困难的勇气和坚强的意志。

  3. 不排斥借助外力。要学会向他人寻求帮助，获得他人的支持和鼓励。

# 4.3 绽放生命之花

# 4.3.1 探索生命的意义

# ★1. 怎样的一生值得过

  1. 活出自我：能够活出自己的人生，实现自我价值。

  2. 帮助他人：在别人需要时付出自己的爱心，无论事情大小，都能承担起自己的责任。

  3. 奉献社会：将个人理想与国家发展、民族复兴和人类命运共同体结合起来。

# 2. 怎样发现和创造生命的意义

  1. 探索生命意义，是生命的原动力之一。
  2. 生命是独特的，生命意义是具体的，要在生活经历中一点点构建自己。
  3. 生命的意义需要自己发现和创造。

# 4.3.2 活出生命的精彩

# ★1. 怎样活出生命的精彩

  1. 对自己：拒绝贫乏，充盈生命。

  2. 对他人：拒绝冷漠，关切他人。

  3. 在平凡中创造伟大。

# ★2. 怎样认识生命的 “平凡与伟大”

  1. 伟大在于创造和贡献。
  2. 平凡的生命也可以伟大。
  3. 当我们将个体生命和他人的、集体的、民族的、国家的甚至人类的命运联系在一起时，生命便会从平凡中闪耀出伟大。

# 5 青春

# 5.1 青春邀约

# 5.1.1 青春变化

# ~1. 进入青春期身体变化的主要表现

身体外形的变化、内部器官的完善、性机能的成熟。

# ~2. 青春期带来的影响

# 积极

青春期的生理变化带来旺盛的生命力，使我们的身体充满能量，拥有充沛的精力，敏捷的思维，对成长充满强烈渴望，感觉生活拥有无限可能。

# 消极

一些正常的生理现象也可能带来烦恼。

# 3. 如何认识和对待青春期的生理变化

  1. 正视身体的变化；
  2. 不因自己的生理变化而自卑，是对自己的尊重；
  3. 不嘲弄同伴的生理变化，是对同伴的尊重；
  4. 在追求形体、仪表等外在美的同时，也要提高品德和文化修养，体现青春的内在美。

# ~4. 怎样正确面对和处理心理矛盾

  1. 青春期的矛盾心理有时让我们烦恼，但也为成长提供了契机，要正确面对和处理。
  2. 参加集体活动、求助他人、培养兴趣爱好转移注意力、学习自我调节。

# 5.1.2 思维成长

# 1. 独立思考的含义与要求

  1. 不等同于一味追求独特。
  2. 不人云亦云。
  3. 有自己独到的见解。
  4. 能接纳他人合理、正确的意见。

# ★2. 思维的批判性的表现

  1. 对事有自己的看法；
  2. 敢于发表不同观点；
  3. 敢于对不合理的事说 “不”；
  4. 敢于挑战权威。

# ★3. 培养思维批判性的意义

  1. 有助于发现和提出问题，从不同角度思考问题，探索解决方案。
  2. 能调动经验，激发新的学习动机，促使我们解决问题，改进现状。

# 4. 培养思维批评性的要求

  1. 要有质疑的勇气，有表达观点、提出合理化建议的能力；
  2. 考虑他人感受，知道怎样批判更容易被人接受，更有利于解决问题；
  3. 掌握批判的技巧。

# ★5. 怎样做勇于创造的好少年，怎样开发创造潜力

  1. 在遵守法律和道德规范的前提下，敢于打破常规，追求创新。
  2. 要关注他人与社会，看重创造的意义和价值。
  3. 创造离不开实践，社会实践是创造的源泉。

生理方面、性格特征、兴趣爱好、思维方式。

# ~2. 了解性别差异、对性别角色的认识的作用

  1. 可以了解自己与异性的不同特点，学会塑造自我形象、与异性相处；

  1. 由国家制定或认可。

  3. 对全体社会成员具有普遍约束力。

  1. 公民在法律面前一律平等，任何人都没有超越法律的特权。

  1. 规范作用：法律规定我们应该享有的权利，应该履行的义务。
  2. 保护作用：法律通过解决纠纷和制裁违法犯罪，惩恶扬善、伸张正义，维护我们的合法权益。

# ★ 5. 自由和规则的关系

  1. ** 社会规则划定了自由的边界。** 自由不是随心所欲，它受到道德、纪律、法律等社会规则的约束。
  2. 公民在行使自由和权利时，不得损害国家的、社会的、集体的利益和其他公民的合法的自由和权利。
  3. ** 社会规则是人们享有自由的保障。** 违反规则、扰乱社会秩序的行为应当受到相应的处罚。

# ★ 6. 如何遵守、维护、改进规则

# 1. 自觉遵守规则
  1. 他律和自律。
  2. 发自内心敬畏规则，将规则作为行动的准绳，内化于心、外化于行。
# 2. 坚定维护规则
  1. 从自己做起，自觉遵守社会规则。
  2. 在保护自身安全的基础上，提醒、监督、帮助他人遵守规则。
# 3. 积极改进规则
  1. 规则不是一成不变的，有的需要制定、废除、修改和完善。
  2. 要积极参与规则的的改进和完善，为新规则的形成建言献策。

# 10.2 道德

# 10.2.1 尊重他人

# ★ 1. 尊重的意义

  1. 对自己：尊重他人是内在修养的表现。
  2. 对他人：每个人都是有尊严的个体，都希望得到他人和社会的尊重。
  3. 对社会：尊重使社会生活和谐融洽。

# ★ 2. 如何尊重他人

  1. 尊重从我做起。积极关注、重视他人。
  2. 平等对待他人。
  3. 学会换位思考。
  4. 学会欣赏他人。

# 10.2.2 以礼待人

# ★ 1. 为什么要以礼待人

  1. 对个人：文明有礼是人立身处世的前提。
  2. 对社会：** 文明有礼促进社会和谐。** 有利于形成安定有序、文明祥和的社会。
  3. 对国家：文明有礼体现国家形象。

# ★ 2. 如何做文明有礼的人

  1. 态度谦和，用语文明。
  2. 举止端庄、文明。
  3. 在社会生活中不断学习、观察、思考和践行，从小事做起，努力做一个懂礼、明礼、守礼的人。

# 10.2.3 诚实守信

# 1. 诚信是什么

诚信是诚实、守信用，是社会主义核心价值观在公民个人层面的一个价值准则，是中华民族的传统美德，是一项民法原则。

# ★ 2. 诚信的重要性

  1. 个人：诚信是人的安身立命之本，是中华民族的传统美德，是融入社会的 “通行证”。
  2. 企业：诚信是企业的无形资产。
  3. 民族：诚信促进社会文明、国家兴旺。

# ★ 3. 如何践行诚信

  1. 树立诚信意识。信守承诺，主动承担责任。
  2. 运用诚信智慧。既恪守诚实的要求，又尊重他人隐私。
  3. 珍惜个人的诚信记录。

# 10.3 守法

# 10.3.1 违法行为

# ★ 1. 含义

违法行为是指出于过错违反法律、法规的规定，危害社会的行为。

# ★ 2. 类别

  1. 民事违法行为（侵犯他人民事权利或没有依法履行合同的义务）
  2. 行政违法行为（如：谎报险情、破坏铁路封闭网、殴打他人等行为都是违反治安管理处罚法的行政违法行为。）
  3. 刑事违法行为（犯罪）

民事违法行为、行政违法行为对社会的危害相对较轻微，称为一般违法行为。

刑事违法行为（犯罪）是最严重的一种违法行为。

违法行为都要承担法律责任。

# 10.3.2 预防犯罪

# ★ 1. 什么是刑法

刑法是惩治犯罪、保护国家和人民利益的有力武器，明确规定了什么行为是犯罪，以及对犯罪应当判处什么样的刑罚。

# ★ 2. 什么是犯罪

犯罪是具有严重的社会危害性、触犯了刑法、应当受到刑罚处罚的行为。

基本特征：严重危害性、刑事违法性、刑罚处罚性。

# 3. 什么是刑罚

犯罪的法律后果是刑罚，是最严厉的强制性法律制裁方法，分为主刑和附加刑。

# ★ 4. 加强自我防范，预防违法犯罪

  1. 珍惜美好生活，认清犯罪危害，远离犯罪。
  2. 杜绝不良行为。
  3. 生活在法治社会，应增强法治观念，依法自律，做一个自觉守法的人。要从小事做起，避免沾染不良习气，自觉遵纪守法，防患于未然。

# 10.3.3 善用法律

# ★ 1. 遇到侵害依法求助的途径

  1. 遇到法律问题或权益受到侵害时，要及时寻求法律援助，依靠法律维护自己的合法权益。
  2. 通过法律服务机构来维护合法权益，如法律服务所、律师事务所、公证处、法律援助中心等。
  3. 受到非法侵害，可寻求国家的法律救济，如依法到公安机关、人民法院、人民检察院控告举报，还可向人民法院提起诉讼。

# 2. 诉讼

诉讼是处理纠纷、应对侵害最正规、权威、有效的手段，是维护合法权益的最后屏障。

诉讼分为：民事诉讼、行政诉讼（“民告官”）、刑事诉讼。

# ★ 3. 如何应对违法犯罪

  1. 同违法犯罪行为作斗争，是包括青少年在内的全体公民义不容辞的责任。
  2. 面对违法犯罪时，要善于斗争，在保全自己、减少伤害的前提下，巧妙地借助他人或社会的力量，采取机智灵活的方式，同违法犯罪做斗争。
  3. 要积极弘扬社会主义法治精神，形成守法光荣、违法可耻的观念，做到自觉守法、遇事找法，解决问题靠法，努力做社会主义法治的忠实崇尚者、自觉遵守者、坚定捍卫者。

# 11 社会责任

# 11.1 责任与角色

# ~1. 责任的含义和来源

责任是一个人分内该做的事，来自对他人的承诺、职业要求、道德规范、法律规定等。

# ~2. 角色与责任的关系

在社会生活的舞台上，我们扮演不同的角色，每种角色都意味着承担相应责任。

# 3. 承担责任的表现

  1. 作为社会的一员，每个人首先要对自己负责。
  2. 要学会感恩，主动关心、帮助和服务他人，在辛苦付出中体验对他人负责的快乐和幸福。

# ★4. 承担责任的意义

  1. 个人角度：只有对自己负责，才能充分挖掘、发挥潜能，才有资格、能力、信心承担起时代和国家所赋予的使命。
  2. 他人角度：只有人人具有责任心，自觉履行应尽的责任，才能共享更加幸福美好的生活。
  3. 社会角度：只有人人认识到自己扮演的角色、承担应尽的责任，才能构建各尽其责、各得其所而又和谐相处的社会。

# 5. 承担责任的代价与回报

  1. 代价：付出时间、精力和金钱，可能因做的不好而受责备甚至处罚。
  2. 回报：承担责任往往伴随着获得物质、精神回报的权利。

# ★6. 怎样做负责任的人

  1. 无论是否自愿选择，都应自觉承担相应责任，不抱怨，不懈怠，全身心投入，我承担我无悔。
  2. 履行社会责任不计代价和回报，敢于承担责任、敢于担当。
  3. 努力提升自身素质，增强履行责任的能力。

# 11.2 奉献社会

# 11.2.1 关爱他人

# ★1. 关爱他人的意义

  1. 他人：关爱传递着美好情感，给人带来温暖和希望，是维系友好关系的桥梁。
  2. 社会：关爱是社会和谐稳定的润滑剂和正能量，有利于形成良好的人际氛围，促进社会文明进步。
  3. 自己：关爱他人，收获幸福。

# ★2. 关爱他人的艺术

  1. 心怀善意。道义上支持、物质上帮助、精神上关怀。
  2. 尽己所能，不分大小，贵在爱心，尽己所能帮助他人，奉献社会。
  3. 讲究策略，考虑他人的内心感受，增强自我保护。

# 11.2.2 服务社会

# ★1. 服务社会对个人成长的意义

  1. ** 体现人生价值。** 只有积极为社会做贡献，才能得到人们的尊重和认可，实现自身价值。
  2. ** 能促进全面发展。** 在服务社会过程中视野不断拓展，能力不断提升，境界不断提高。

# ★2. 奉献社会的途径

  1. ** 青年担当责任。** 在新时代，共青团要组织青年参加改革开放和社会主义现代化建设的实践。
  2. 积极参与社会公益活动。
  3. 热爱劳动，爱岗敬业。

# 12 维护国家利益

# 12.1 国家利益

# ~1. 内涵及外延

  1. 国家利益是一个主权国家在国际社会中生存需求和发展需求的总和，包括人口、领土、主权和政权等，关系民族生存、国家兴亡。
  2. 国家利益涉及政治、军事、经济、文化、社会等领域，包括安全利益、政治利益、经济利益、文化利益等。
  3. 国家的核心利益包括国家主权、国家安全、领土完整、国家统一、宪法确立的国家政治制度和社会大局稳定、经济社会可持续发展的基本保障。

# ★2. 与人民利益的关系

  1. 国家利益反映广大人民的共同需求，是人民利益的集中表现，国家利益至上，人民利益高于一切，二者相辅相成。
  2. 维护人民利益离不开国家利益，人民利益只有上升、集中到国家利益，运用国家的工具，才能得到真正的维护。
  3. 国家利益的实现离不开人民，国家利益只有反映人民利益，依靠人民艰苦奋斗，才能得到真正的实现。
  4. 在当代中国，国家利益和人民利益是高度统一的。实现中华民族伟大复兴就是将国家和人民视为命运共同体，将二者利益紧密联系在一起。

# ★3. 树立维护国家利益意识

  1. 心怀爱国之情，牢固树立国家利益至上的观念，以热爱祖国为荣，以危害祖国为耻。
  2. 树立和增强危机意识和防范意识，对危害国家利益、威胁国家生存和发展的行为时刻保持警惕。
  3. 增强维护国家利益的责任感和使命感。用理性、务实、文明的心态合法有序地表达爱国情感，维护国家利益。

# ★4. 如何捍卫国家利益

  1. 无论何时何地，都应着眼长远、顾全大局，以国家利益为重，把国家利益放在第一位。
  2. 为了国家利益，有时不仅需要放弃个人利益，甚至要献出自己的生命。
  3. 捍卫国家尊严，坚决同一切损害国家利益的行为作斗争。日常生活中自觉遵守道德和纪律，积极维护国家稳定团结的局面。

# 12.2 国家安全

# 1. 国家安全的重要性

国家安全与我们息息相关，是国家生存和发展的重要保障，是人民幸福安康的前提。

# ~2. 如何坚持总体国家安全观

  1. 以人民安全为宗旨，以政治安全为根本，以经济安全为基础，以军事、文化、社会安全为保障，以促进国际安全为依托，走出一条中国特色国家安全道路。
  2. 面对国家安全形势的新变化，要构建集政治、国土、军事、经济、文化、社会、科技、信息、生态、资源、核安全等于一体的国家安全体系。

# ★3. 如何维护国家安全

# 国家

全面推进国防和军队现代化，坚持党对人民军队的绝对领导。

# ★个人，如何履行维护国家安全的义务

维护国家安全，人人可为。

  1. 维护国家安全是我们的共同责任。要增强国家安全意识，树立国家安全利益高于一切的观念，提高防范意识。
  2. 通过各种方式为维护国家安全贡献智慧和力量。
  3. 认真学习有关国家安全和保密工作的法律法规、规章制度。
  4. 每个公民都应严格遵守有关国家安全的法律规定，积极履行维护国家安全的法定义务，不断提高防范能力。

# 12.3 建设美好祖国

# 1. 劳动的意义

** 劳动是财富、幸福的源泉。** 我国每一项成就的取得，都是广大人民群众用辛勤劳动、诚实劳动、创造性劳动换来的，中国人民用实干精神创造了今天的辉煌。

# 2. 如何对待不同岗位的劳动者

  1. 每个人所处的岗位不同，从事不同的劳动，但都在为国家和社会做贡献。
  2. 正是无数劳动者艰苦奋斗才成就了今天的美好生活。
  3. 无论是脑力劳动者还是体力劳动者，都是和国家的建设者，都值得尊敬和学习。

# 3. 青少年发扬实干精神

# 原因

中国梦的实现需要一代代人埋头苦干和接力奋斗，需要每个人在各自岗位上付出辛勤和汗水。只有继续发扬实干精神，才能创造新的辉煌。

# 做法

接过历史的接力棒，继续发扬实干精神，努力学习，积极探索，（以执着的信念、优良的品德、丰富的知识、过硬的本领，）担负起历史重任。
